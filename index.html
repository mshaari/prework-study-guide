HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Things like h1, h2, h3, ..., h7 represent the level of a heading the given text block represents

CSS

  • CSS (Cascading Style Sheets) is used to give the content style
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The box-shadow CSS property adds shadow effects around the element’s frame. The X offset is 5px (shadow to the right of the box). The Y offset is 10px (shadow under the box).

Git

  • Git and GitHub are version control software and cloud hosting services used to track changes in code and store changes accessible to everyone.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used to handle logic and allow users to interact with a website.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A group of related data is an array and for loops perform tasks on each piece of data.
  • A function is a set of instructions that tells the computer how to perform a certain task.
